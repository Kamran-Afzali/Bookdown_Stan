Chapter 4 Bayesian Modeling in R and Stan: Mediation Analysis

In the previous chapter, we extended Bayesian regression in Stan to handle categorical predictors and interactions. In this chapter, we turn to mediation analysis, where we ask whether the effect of a predictor \(X\) on an outcome \(Y\) operates partly through an intermediate variable \(M\). The key quantities of interest are the direct effect of \(X\) on \(Y\), the indirect effect of \(X\) on \(Y\) through \(M\), and the total effect. pmc.ncbi.nlm.nih

4.1 The Mediation Setup

We consider the classic single-mediator model:

  • \(X\): the predictor or treatment.
  • \(M\): the mediator.
  • \(Y\): the outcome.

The mediation logic is simple: \(X\) may influence \(M\), and \(M\) may in turn influence \(Y\). If \(X\) also affects \(Y\) directly, then the total effect can be decomposed into a direct path and an indirect path through the mediator. In a linear Gaussian model, this decomposition is especially transparent because the indirect effect is the product of two regression coefficients. research-portal.uu

4.2 Simulating the Data

To make the idea concrete, we simulate data from a mediation model with one predictor, one mediator, and one outcome.

set.seed(42)

n <- 500

# Predictor
x <- rnorm(n, 0, 1)

# True parameters
a0 <- 0
a1 <- 0.8

b0 <- 0
b1 <- 1.2
c_prime <- 0.5

sigma_m <- 1.0
sigma_y <- 1.5

# Mediator
m <- a0 + a1 * x + rnorm(n, 0, sigma_m)

# Outcome
y <- b0 + c_prime * x + b1 * m + rnorm(n, 0, sigma_y)

data <- data.frame(x, m, y)
head(data)

In this data-generating process, the effect of \(X\) on \(M\) is \(a_1\), the effect of \(M\) on \(Y\) is \(b_1\), and the direct effect of \(X\) on \(Y\) is \(c'\). The indirect effect is \(a_1 b_1\), and the total effect is \(c' + a_1 b_1\). davidakenny

4.3 The Stan Model

We fit the mediation model as two linked regressions: one for the mediator and one for the outcome. The mediator model estimates how \(X\) predicts \(M\), and the outcome model estimates how \(X\) and \(M\) jointly predict \(Y\).

stan_mod_med <- "
data {
  int<lower=1> N;
  vector[N] x;
  vector[N] m;
  vector[N] y;
}

parameters {
  real a0;
  real a1;
  real b0;
  real b1;
  real c_prime;
  real<lower=0> sigma_m;
  real<lower=0> sigma_y;
}

model {
  a0 ~ normal(0, 10);
  a1 ~ normal(0, 10);
  b0 ~ normal(0, 10);
  b1 ~ normal(0, 10);
  c_prime ~ normal(0, 10);
  sigma_m ~ exponential(1);
  sigma_y ~ exponential(1);

  m ~ normal(a0 + a1 * x, sigma_m);
  y ~ normal(b0 + c_prime * x + b1 * m, sigma_y);
}

generated quantities {
  real indirect = a1 * b1;
  real total = c_prime + indirect;
}
"

writeLines(stan_mod_med, con = "stan_mod_med.stan")
cat(stan_mod_med)

This parameterization is useful because the mediation effects are defined directly in terms of the regression coefficients. Stan then gives us posterior draws of \(a_1\), \(b_1\), and \(c'\), from which we can compute the indirect and total effects inside generated quantities. mc-stan

4.4 Fitting the Model

We now fit the model with rstan.

library(rstan)

stan_data_med <- list(
  N = nrow(data),
  x = data$x,
  m = data$m,
  y = data$y
)

fit_med <- rstan::stan(
  file   = "stan_mod_med.stan",
  data   = stan_data_med,
  iter   = 2000,
  warmup = 1000,
  chains = 4,
  seed   = 42
)

print(fit_med, pars = c("a0", "a1", "b0", "b1", "c_prime", "sigma_m", "sigma_y", "indirect", "total"))

The posterior summaries should recover the true values closely if the model is well specified. Because the model is Bayesian, the direct and indirect effects are estimated with full uncertainty, not just point estimates. pmc.ncbi.nlm.nih

4.5 Interpreting Effects

The mediation quantities are interpreted as follows:

Quantity Meaning
Direct effect Effect of \(X\) on \(Y\) not through \(M\).
Indirect effect Effect of \(X\) on \(Y\) that passes through \(M\).
Total effect Sum of direct and indirect effects.

In this linear model, the indirect effect is \(a_1 b_1\), while the direct effect is \(c'\). A useful posterior summary is the probability that the indirect effect is positive, which can be computed as mean(extract(fit_med)$indirect > 0). healthsci.queensu

4.6 Posterior Effect Estimates

A clean way to report mediation results is to summarize the posterior distribution of each effect.

post <- rstan::extract(fit_med)

indirect_draws <- post$indirect
total_draws <- post$total
direct_draws <- post$c_prime

c(
  direct_mean = mean(direct_draws),
  direct_ci_l = quantile(direct_draws, 0.025),
  direct_ci_u = quantile(direct_draws, 0.975),
  indirect_mean = mean(indirect_draws),
  indirect_ci_l = quantile(indirect_draws, 0.025),
  indirect_ci_u = quantile(indirect_draws, 0.975),
  total_mean = mean(total_draws),
  total_ci_l = quantile(total_draws, 0.025),
  total_ci_u = quantile(total_draws, 0.975)
)

This gives you posterior means and credible intervals for all three effects. Because these are posterior draws, the intervals have a direct probabilistic interpretation: they describe where the parameter likely lies given the model and data. research-portal.uu

4.7 Posterior Diagnostics

As in any Bayesian workflow, we should check convergence and fit before interpreting the effects. Trace plots and \(\widehat{R}\) values help diagnose sampling issues, while posterior predictive checks tell us whether the model reproduces the observed data reasonably well. mc-stan

library(bayesplot)

fit_med %>%
  mcmc_trace(pars = c("a1", "b1", "c_prime", "indirect", "total"))

For posterior predictive checks, it is often helpful to compare replicated outcome distributions against the observed outcome distribution. Stan’s posterior predictive workflow is designed exactly for this kind of model checking. mc-stan

4.8 Why Bayesian Mediation Helps

Bayesian mediation analysis is particularly useful because the indirect effect is often skewed and does not sit comfortably within normal approximations; posterior simulation instead yields the full distribution of the indirect effect, avoiding reliance on delta-method approximations or large-sample assumptions while naturally propagating uncertainty through nonlinear quantities such as (a_1 b_1). This is largely why mediation aligns well with probabilistic programming frameworks like Stan. Conceptually, mediation in this setting is straightforward once framed as two linked regression models: the direct and indirect effects are not explicit parameters in the likelihood but are derived from posterior draws of regression coefficients, providing a flexible and fully Bayesian characterization of the mediation pathway. From there, the framework extends quite naturally—one may introduce covariates into both mediator and outcome models, accommodate multiple mediators, move to multilevel structures with varying intercepts or slopes, or consider non-Gaussian outcomes such as binary or ordinal responses with appropriate likelihoods. A natural progression is toward hierarchical mediation models, where the mediation pathway itself varies across groups, which also creates a useful conceptual bridge to earlier discussions of group-specific effects and interactions econtent.hogrefe.
